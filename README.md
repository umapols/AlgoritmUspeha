# Форма «Сотрудник предприятия» (ООО «Алгоритм Успеха»)

## Описание
Простое Windows Forms приложение для учёта сотрудников веб-студии. Позволяет добавлять, редактировать, удалять данные о сотрудниках, а также выполнять валидацию полей (ФИО, паспортные данные, дата рождения и др.).

## Технологии
- Язык: C# (.NET Framework 4.7.2)
- Тип приложения: Windows Forms
- База данных: не используется (данные хранятся в памяти для демонстрации)

## Функциональные возможности
- Добавление нового сотрудника
- Редактирование существующего
- Удаление сотрудника
- Валидация полей:
  - ФИО (только кириллица)
  - Серия паспорта (4 цифры, не 0000)
  - Номер паспорта (6 цифр)
  - Дата рождения (не в будущем, не раньше 1900 г.)
- Загрузка фотографии сотрудника
