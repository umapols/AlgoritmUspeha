1. Общие сведения
Название проекта: Автоматизированная информационная система «Сотрудник предприятия»
Краткое название: AlgoritmUspeha
Автор: Кирьянов А.В.
Организация: ООО «Алгоритм Успеха»
Дата начала разработки: 08.06.2026
Текущая версия: 1.0

2. Назначение проекта
Система предназначена для автоматизации процесса учета личных и профессиональных данных сотрудников ИТ-отдела.

3. Основные функции
- Регистрация и хранение данных о сотрудниках (ФИО, фотография, пол, дата рождения, паспортные данные и т.д.)
- Просмотр списка всех сотрудников.
- Редактирование и удаление существующих записей.
- Валидация вводимых данных (проверка корректности паспортных данных, дат и т.п.).
- Поиск сотрудника по фамилии.

4. Основные характеристики
- Технологии: C#, Windows Forms.
- Архитектура: клиент-серверная (клиентское приложение с локальной базой данных).
- Требования к ПО: Операционная система Windows 10/11, установленный .NET Framework.

5. Инструкция по запуску (кратко)
1. Клонировать репозиторий: git clone [ссылка_на_ваш_репозиторий]
2. Открыть файл решения (.sln) в Visual Studio.
3. Нажать кнопку «Запуск» (F5) для компиляции и запуска приложения.
