Рекомендации по работе с Git для проекта:

Рекомендация 1: Всегда создавайте новую ветку для новой функциональности.
Рекомендация 2: Называйте ветки осмысленно (например, 'feature/add-validation').
Рекомендация 3: Перед созданием ветки `extra` убедитесь, что вы находитесь в основной ветке (`main`).
Рекомендация 4: Клонируйте репозиторий в Visual Studio для удобной работы с кодом.
Рекомендация 5: Регулярно делайте коммиты с понятными сообщениями.
